Implementation Pattern

Instead of using inline Tailwind classes, use the Typography components:

// Before: Inline classes repeated everywhere <h1 className="text-2xl font-bold mb-3 text-gray-900 dark:text-white"> Title </h1> <p className="text-md text-gray-500 dark:text-gray-400 leading-relaxed mb-6"> Description </p> // After: Reusable Typography components <H1>Title</H1> <Paragraph size="lg">Description</Paragraph>
